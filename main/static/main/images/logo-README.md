# Logo de l'Église

## Instructions pour le logo

Pour afficher le logo de votre église dans la barre de navigation :

1. **Nom du fichier** : `logo1.jpg` (ou `logo1.png`)
2. **Emplacement** : Placez votre image dans ce dossier : `main/static/main/images/`
3. **Dimensions recommandées** : 200x200 pixels minimum (format carré)
4. **Format** : JPG ou PNG

## Caractéristiques du logo

- **Forme** : Le logo sera affiché en cercle (40x40 pixels)
- **Position** : À gauche de "Église" dans la barre de navigation
- **Effet** : Légère animation au survol (agrandissement)
- **Bordure** : Bordure blanche semi-transparente

## Fallback

Si aucune image `logo1.jpg` n'est trouvée, le logo ne s'affichera pas et seul le texte "Église" sera visible.

## Formats supportés

- `logo1.jpg` (priorité)
- `logo1.png` (alternative)
- `logo1.jpeg` (alternative)
