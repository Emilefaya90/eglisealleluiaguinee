# Dossier Images

## Structure des dossiers

- **hero/** : Images de fond pour la page d'accueil
  - Placez ici l'image principale qui s'affichera en arrière-plan
  - Formats recommandés : JPG, PNG, WebP
  - Résolution recommandée : 1920x1080 ou plus

- **gallery/** : Images de la galerie
  - Placez ici les images supplémentaires pour la galerie
  - Formats recommandés : JPG, PNG, WebP
  - Résolution recommandée : 800x600 ou plus

## Instructions d'utilisation

1. Copiez vos images dans les dossiers appropriés
2. Nommez l'image principale : `4.jpg` (ou .png/.webp)
3. Nommez les images de galerie : `gallery-1.jpg`, `gallery-2.jpg`, etc.
4. L'application affichera automatiquement les images disponibles

## Exemples de noms de fichiers

- `hero/4.jpg` - Image de fond principale
- `gallery/gallery-1.jpg` - Première image de galerie
- `gallery/gallery-2.jpg` - Deuxième image de galerie
- `gallery/gallery-3.jpg` - Troisième image de galerie
